Detailed Description

The general notation:
    - L_ is a label
    - cB_ is a column bar
    - lE_ is a line Edit
    - gL_ is a grid Layout
    - tE_ is a text Edit
    - sB_ is a scroll Bar
    - B_ is a bottom
    - lW_ is a (layout) QWidget
